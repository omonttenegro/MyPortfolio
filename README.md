# André Montenegro Portfolio

A personal portfolio website built to showcase my projects, experience, and interests in software development, artificial intelligence, and computer science.

## About the Project

This website works as a central hub for my academic and technical work. It was designed to present my profile in a clear and structured way, combining personal information, selected projects, and relevant experience in one place.

The portfolio is organized into five main sections:

- **Home**
- **About**
- **Experience**
- **Projects**
- **Contact**

## What the Portfolio Includes

### Personal Presentation
A concise introduction to who I am, my background, and my areas of interest.

### Experience
A section dedicated to relevant experiences, including:
- Erasmus in Krakow
- CS50 coursework and related development work

### Selected Projects
A curated set of projects in AI, algorithms, and software engineering, including:
- Tic-Tac-Toe
- Minesweeper AI
- PageRank
- Traffic Sign Classification
- Attention
- Music Collection Database System
- 3D Printer Lab Management System
- Parking Space Allocation Manager

### Contact
A simple way to reach out professionally.

## Objective

The goal of this portfolio is to provide a clean, modern, and accessible overview of my work, while also serving as a public-facing project hub connected to my GitHub repositories.

## Current Status

This project is still in development and will continue to be updated as new work, experience, and refinements are added.
