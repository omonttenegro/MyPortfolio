Project spotlight

AI Minesweeper

Originally built in Python for CS50 AI, this project uses knowledge-based reasoning, sentence inference, safe-move selection, and fallback random moves to solve Minesweeper.

Knowledge-based AI
Sentence inference
Safe and random move logic
View Code Back to projects

Core ideas

  • Each revealed clue creates a sentence over nearby hidden cells.
  • The AI infers safe cells and known mines from the knowledge base.
  • When logic is exhausted, it falls back to a random move.

Controls

  • Left click to reveal a cell.
  • Right click or use flag mode to place flags.
  • Use AI Step or Autoplay to watch the reasoning engine in action.
